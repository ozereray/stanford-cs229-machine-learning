### üìÇ Lecture Archive (Full Curriculum)

The following 20 lectures provided the core technical framework for this repository:

#### **Module 1: Supervised Learning**
* **Lectures 01-02:** Linear Regression, Gradient Descent, and Normal Equations.
* **Lectures 03-04:** Logistic Regression, Newton's Method, and Generalized Linear Models (GLM).
* **Lecture 05:** Generative Learning Algorithms and Gaussian Discriminant Analysis (GDA).
* **Lectures 06-08:** Support Vector Machines (SVM), Lagrange Duality, and the Kernel Trick.

#### **Module 2: Learning Theory**
* **Lectures 09-11:** Bias-Variance Tradeoff, VC Dimension, Uniform Convergence, and Model Selection.

#### **Module 3: Unsupervised Learning**
* **Lectures 12-13:** K-Means Clustering and the Expectation-Maximization (EM) Algorithm.
* **Lectures 14-15:** Principal Component Analysis (PCA), Factor Analysis, and Independent Component Analysis (ICA).

#### **Module 4: Reinforcement Learning & Optimal Control**
* **Lectures 16-17:** Markov Decision Processes (MDP), Value Iteration, and Policy Iteration.
* **Lecture 18:** Linear Quadratic Regulation (LQR) and Riccati Equations.
* **Lecture 19:** Kalman Filters, LQG Control, and the Separation Principle.
* **Lecture 20:** Policy Search, REINFORCE, and the PEGASUS Algorithm.
